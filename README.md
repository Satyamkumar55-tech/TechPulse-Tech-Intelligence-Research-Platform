# AI-Powered Tech News Researcher

An intelligent web application that curates, analyzes, and presents the latest technology news using AI-driven workflows. Built with **React + Vite**, the project emphasizes speed, modularity, and clean UI/UX design.

---

## 🚀 Features
- **AI-Powered Summaries** – Automatically condenses long tech articles into digestible insights.  
- **Categorized Views** – Organized sections for assets, components, data, and views.  
- **Responsive UI** – Modern design with reusable React components.  
- **Workflow Automation** – Custom workflows for fetching, processing, and displaying content.  
- **Optimized Build** – Fast bundling and hot-reload via Vite.

## ⚙️ Installation & Setup
1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/AI-Tech-News-Researcher.git
   cd AI-Tech-News-Researcher
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
